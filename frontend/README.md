
# Don Bosco School Thakurnagar Website – Official Repository

This repository contains the complete source code for the official website of Don Bosco School Thakurnagar.  
The website is developed to provide students, parents, teachers, and visitors with clear, reliable, and up-to-date information about the school.

---

## About the Project

The school website acts as the school’s digital identity and information hub.  
It presents academic details, co-curricular activities, school facilities, events, and important announcements in a structured and user-friendly manner.

The website is designed with a modern layout while maintaining simplicity and accessibility for all users.

---

## Technology Stack

- **Frontend Framework:** React.js  
- **Build Tool:** Vite  
- **Styling:** Tailwind CSS  
- **Animations:** Framer Motion  
- **Version Control:** Git & GitHub
